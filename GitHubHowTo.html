¿Cómo volcar tu repositorio local en GitHub desde terminal?

Necesitas:

  • Descargar Git
  • Abrir una terminal
  • Navegar hasta el directorio donde quieras descargar el repo (cd)

git clone https://github.com/mariasanchezc/angular-coursera.git

Editas lo que quieras en el editor que prefieras (atom, brackets, sublime text, visual studio, etc.)

Te puedes ayudar de browser-sync para visualizar en tiempo real los cambios que le hagas al html

browser-sync start --server --directory --files "**/*"

Comando de git:

  • Comprobar estado de git:
    git status
  • Incluir archivos editados en local (en el repo donde estemos situados) antes de ser guardados/commiteados:
    git add .
  • Commitear con un comentario:
    git commit -m "mensaje que resume el commit"
  • Commitear añadiendo identidad
    git config --global user.email "you@example.com"
    git config --global user.email "Your Name"
  • Pushear un commit que tengamos en local:
    git push
    (+ logearnos en gitHub en el pop-up que salta)
